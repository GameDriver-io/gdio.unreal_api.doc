# ApiClient.SetObjectPosition method

Easily set the position of an AActor or USceneComponent in world coordinates.

```csharp
public bool SetObjectPosition(string hierarchyPath, Vector3 newPosition, int timeout = 10)
```

| parameter | description |

| newPosition | The position to move the AActor or USceneComponent to. |
