# ApiClient.Scroll method

Use this function to simulate mouse wheel scrolling.

```csharp
public bool Scroll(float x, float y, ulong frame_count = 1, int timeout = 30)
```

| parameter | description |
| --- | --- |
| x | The x value of the scroll vector for the mouse, usually 0. |
| y | The y value of the scroll vector for the mouse, typically -1 &lt;= y &lt;= +1 |
| frame_count | The numer of frames to activate scroll event. normally 1 |
| timeout | The time to wait for the agent to process the Scroll request. |

## Return Value

TRUE if the GameDriver agent was able to successfully process the request.

## Examples

```csharp
//Move the mouse to a location then scroll.
            api.MouseMoveToPoint(new Vector2(1491, 350), 90, new Vector2(1260, 610));
             api.WaitForEmptyInput();
             api.Scroll(0, -1, 1);
             api.Scroll(0, -1, 1);
             api.Scroll(0, -1, 1);
```
